# BookTracker

Group1's Book Tracker — PHP + MySQL application deployed on AWS.

**Course:** CMP6210 Cloud Computing (2025-6), Birmingham City University
**Author:** Abdullah Hafidz | **Group:** Group 1

## Stack

- PHP 8.x (custom MVC, no framework)
- MySQL 8.x (PDO + prepared statements)
- Apache with mod_rewrite
- AWS: EC2, RDS Multi-AZ, ALB, S3, CloudFront, CloudWatch, CloudTrail

## Local development

Requires Laragon (Apache + PHP + MySQL). Access at `http://localhost/booktracker`.

## AWS deployment

See the deployment guide in the dev repository for full step-by-step instructions.

Region: `ap-southeast-1` (Singapore)
